# 🚀 JobTrackr v2

An all-in-one Job Application Tracking System designed to organize your job search journey, keep track of interviews, manage documents, and store interview insights in one place.

## 🌐 Live Demo
Experience the application here:
👉 [https://job-tracker-eight-bay.vercel.app](https://job-tracker-eight-bay.vercel.app)

---

## 🛠️ Tech Stack
* **Frontend:** React.js, Vite, Tailwind CSS, Axios, React Router.
* **Backend:** Node.js, Express.js.
* **Database:** MongoDB.
* **Deployment:** Vercel (Frontend), Render (Backend).

## ✨ Key Features
- 📊 **Dashboard:** Get a bird's-eye view of your application statistics, including totals, applied jobs, interviews, and status updates.
- 💼 **Job Management:** Seamlessly add, edit, and delete job applications.
- 📝 **Detailed Tracking:** Add personal notes, interview questions, and insights for every company.
- 📁 **Document Management:** Securely store job-related documents for each application.
- 🔔 **Stay Organized:** Never miss a follow-up with structured data management.
